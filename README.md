# ung-facilities-maps
UNG Public Facilities Maps

Repository Scope & Data Sensitivity
This repository hosts public-facing UNG Facilities GIS content only — campus maps, wayfinding tools, project status pages, and related web assets published via GitHub Pages.
What belongs here:

Public maps, StoryMaps, and wayfinding interfaces
General construction/project status summaries intended for the campus community
Front-end code (HTML/CSS/JS) for public dashboards

What does NOT belong here:

Internal floor plans, room numbering schemes, or CAD/GIS source data
Building security details (camera locations, access control, key systems)
Network/IT infrastructure documentation
Personally identifiable information (names, emails, ticket details) from maintenance or survey submissions
Credentials, API keys, OAuth tokens, or service account information

Internal architecture, pipeline scripts, and administrative documentation are maintained in private repositories (e.g., ung-space-audit, ung-facilities-internal).
Before committing: review new files for embedded personal data, internal addresses beyond public building names, or anything that wouldn't be appropriate on a public-facing university webpage.
