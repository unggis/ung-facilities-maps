## Maintaining These Documents

Markdown source files and the build scripts used to generate these PDFs
are maintained in the private repository:

Source files are maintained in the UNG Facilities GIS internal repository.
Contact the GIS Analyst / Facilities Planning Specialist for access.

To update a document:
1. Edit the `.md` source file in `ung-facilities-internal`
2. Regenerate the PDF using `build_pdf.py` (see `ung-facilities-internal` for instructions)
3. Upload the updated PDF here to replace the existing file
4. Do not move or rename files in this folder — Hub page links point directly to these paths
