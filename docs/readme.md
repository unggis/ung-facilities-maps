
# UNG Facilities GIS — Program Documents

Public-facing GIS program documents served via the UNG Facilities Hub page.
Files in this folder are accessible via direct URL through GitHub Pages.

---

## Documents

| File | Description |
|------|-------------|
| UNG_GIS_UnderstandingMapAccuracy.pdf | Guide to positional accuracy, data tiers, and appropriate use of GIS data |
| UNG_GIS_BenchmarkCapture_Standard.pdf | Workflow and schema for capturing survey control points from construction drawings |
| UNG_GIS_DataDictionary.pdf | Enterprise GDB schema reference — field names, types, and domains |
| USG_FICM_Room_Use_Code_Supplement_2012-10_MODIFIEDJDS.pdf | USG FICM room use code reference, modified for UNG application |
| room_numbering_sop_draft.pdf | SOP-FAC-001 — Room numbering governance (draft pending approval) |

## Source Files

The `source/` subfolder contains the markdown source files used to generate
the PDFs above. PDFs are the distributed versions; markdown files are the
maintained versions. When a document is updated, edit the markdown source
and regenerate the PDF.
