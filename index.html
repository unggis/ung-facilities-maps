UNG Facilities GIS — Map Assets Repository

University of North Georgia · Facilities Management

What this is

A public repository of static map assets, HTML/JS files, and GeoJSON used to power UNG Facilities public-facing GIS displays. All content in this repository is intended for public viewing.

Where to go

For the public-facing GIS portal visit gis-ung-facilities.hub.arcgis.com.

For UNG Facilities Management visit ung.edu/facilities.

Data & security

This repository contains no space inventory data, personnel records, or sensitive institutional information. Enterprise GIS data — including floor plan geometry, room assignments, and Active Directory attributes — is maintained in secured systems accessible only to authorized UNG personnel.
